📖 Apps Script 설치 방법 보기
1. 구글 스프레드시트 새 파일 생성
2. 확장 프로그램 → Apps Script 클릭
3. Code.gs 파일에 제공된 Apps Script 코드 전체 붙여넣기
4. 저장(Ctrl+S) 후 배포 → 새 배포
5. 유형: 웹 앱 / 액세스: 모든 사용자 설정
6. 배포 URL을 위 입력란에 붙여넣기
7. 💾 URL 저장 클릭 후 각 항목 📤 시트에 올리기 클릭
