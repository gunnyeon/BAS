🔗 연동 설정
Apps Script → 배포 → 웹앱 URL을 복사해서 붙여넣으세요
📋 설정 순서
1
Google Sheets 열기 → 확장 프로그램 → Apps Script
2
Code.gs 내용 전체 붙여넣기 후 저장
3
배포 → 새 배포 → 유형: 웹 앱
4
액세스 권한: 전체 (익명 포함) 으로 설정
5
생성된 웹앱 URL을 위 입력창에 붙여넣기
6
저장 및 연결 클릭 → 자동으로 시트 초기화 및 샘플 데이터 삽입
